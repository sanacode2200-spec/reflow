# 令和8年診療報酬改定 疾患別リハビリ 点数リファレンス

> **用途**: ReFlow 帳票機能における点数算定の検証用マスタ。入院・外来の医療保険における疾患別リハビリを対象とする。
> **施行日**: 令和8年（2026年）6月1日
> **出典**: PT-OT-ST.NET 令和8年診療報酬改定特設サイト（厚生労働省告示第69号・保医発0305第6号に基づく）
> **最終更新**: 2026-05-30
>
> ⚠️ この値は帳票の点数自動計算・検証のソースオブトゥルース。改定差分は必ず「旧→新」を併記し、移行期（R8.5.31以前の算定実績の扱い）も明記する。

## 2. 加算（1単位あたり、特記なき場合）

### 2-1. 早期リハビリテーション加算 ★大幅変更

|          | 令和6年（旧）                     | 令和8年（新）                                    |
| -------- | --------------------------------- | ------------------------------------------------ |
| 点数     | 一律 **25点**                     | **入院日から3日目以内: 60点 / 4日目以降: 25点**  |
| 限度期間 | 発症・手術・急性増悪から **30日** | 入院日から起算して **14日**                      |
| 起算日   | 発症・手術・急性増悪              | **入院した日**                                   |
| 転院患者 | （規定なし）                      | **転院前の医療機関に入院した日**を起算日         |
| 外来患者 | （規定なし）                      | **退院前の入院日**を起算日（脳卒中等の対象患者） |

- ⚠️ 心大血管の旧起算日は「発症・手術・急性増悪から7日目又は治療開始日のいずれか早いもの」だった点に注意（旧データ移行時）。

### 2-2. 休日リハビリテーション加算 ★新設

- 点数: **1単位につき25点**
- 限度期間: 発症・手術・急性増悪から **30日目まで**
- 対象の休日: **土曜日・日曜日・祝日**（祝日は国民の祝日法第3条の休日。1/2・1/3・12/29・12/30・12/31も含む）
- ⚠️ 移行措置: 起算日相当日がR8.5.31以前でも、その日を起算日とみなし（リセットしない）、6/1以降の要件を満たす日に算定可能。

### 2-3. 初期加算（変更なし）

- 点数: **1単位につき45点**
- 限度期間: 発症・手術・急性増悪から **14日**
- 早期リハ加算に「更に」加算する形

### 2-4. 急性期リハビリテーション加算（変更なし）

- 点数: **1単位につき50点**
- 限度期間: 発症・手術・急性増悪から **14日**
- 入院患者限定。対象患者要件あり（BI10点以下／認知症自立度ランクM以上／一定の処置等／対象感染症等）

> **加算の積み上げイメージ（入院・脳血管Ⅰの例）**
> 所定点数245 ＋ 早期リハ加算（3日以内60／4日以降25）＋ 初期加算45 ＋ 急性期リハ加算50 ＋（休日なら）休日リハ加算25

## 3. 減算・単位制限 ★新設

### 3-1. 離床を伴わないリハビリの評価見直し

- 離床を伴わずに行う場合の**区分を新設（減算評価）**。5疾患共通。
- 帳票には「離床の有無」フラグが必要。

### 3-2. ベッド上のみリハビリの2単位制限 ★新設

- ベッド上のみのリハビリは**1日2単位まで**の制限規定が新設。
- 全5疾患の変更ポイントに明記。

## 4. リハビリテーション総合計画評価料（H003-2）★段階制へ変更

|                           | 令和6年（旧）           | 令和8年（新） |
| ------------------------- | ----------------------- | ------------- |
| 総合計画評価料1 初回      | 300点                   | **300点**     |
| 総合計画評価料1 2回目以降 | （区分なし・一律300点） | **240点**     |
| 総合計画評価料2 初回      | 240点                   | **240点**     |
| 総合計画評価料2 2回目以降 | （区分なし・一律240点） | **196点**     |

- 算定頻度: 患者1人につき1月に1回
- 「初回」を算定する条件:
  - 転医（転院・退院含む）後、自院で初めて算定する場合
  - 新たな疾患の発症・再発・急性増悪で起算日が再設定され、計画書を改めて作成・評価した場合
- ⚠️ 移行措置: R8.5.31以前に評価料1または2を算定済みなら、6月以降は**「2回目以降の場合」**を算定。

## 5. H003-4 目標設定等支援・管理料 ★廃止

- 旧点数: 初回250点／2回目以降100点（3月に1回、要介護被保険者等が対象）→ **削除**
- 連動して以下も廃止:
  - 脳血管・廃用・運動器リハで、過去3月以内に目標設定等支援・管理料を未算定の要介護被保険者等に対する**所定点数90%減算規定（注）を削除**
- 帳票対応: H003-4の項目・帳票テンプレートを削除。減算フラグの管理も不要に。

## 6. 通則レベルの算定ルール変更（帳票ロジックに影響）

- 計画書の説明主体: 医師＋医師の指示を受けた看護師・PT・OT・STも可（ただし回復期リハ病棟・特定機能病院リハ病棟入院料算定患者は医師に限定）
- **患者等の署名は不要**（旧: 原則署名）。説明日・説明者を計画書写しまたは診療録に記録。旧様式21の使用可が明記。
- 訓練時間に含めない時間: 計画書の作成・説明時間、リハビリ記録時間、移動時間は**訓練時間（1単位20分）に含めない**。
- 医療機関外リハ: 1日3単位＋一連の入院で合計3単位（別表第九の三の患者は6単位）まで追加で疾患別リハとみなせる。
- 1日の算定単位数: 患者1人1日**6単位**（別に定める患者は**9単位**）。運動器リハ（Ⅰ）は9単位対象から削除。

## 7. 帳票機能 実装チェックリスト

点数計算・検証で必ず参照すべき入力項目（フィールド）:

- [ ] 疾患区分（H000/H001/H001-2/H002/H003）
- [ ] 施設基準区分（Ⅰ/Ⅱ/Ⅲ）
- [ ] 実施職種（PT/OT/ST/医師/看護師/集団/それ以外）
- [ ] 入院/外来の別
- [ ] 起算日（疾患別に定義が異なる：治療開始日 or 発症・手術・急性増悪・最初の診断日）
- [ ] **入院日**（早期リハ加算の起算日／転院時は転院前入院日／外来は退院前入院日）
- [ ] 実施日（休日判定＝土日祝、年末年始特例含む）
- [ ] 離床の有無（減算判定）
- [ ] ベッド上のみか（2単位制限）
- [ ] 当日の累計単位数（6/9単位上限、運動器Ⅰは9単位対象外）
- [ ] 総合計画評価料の算定回数（初回/2回目以降）
- [ ] 移行措置フラグ（R8.5.31以前の算定実績有無）

### 点数計算の優先順位（疑似コード）

```
1単位あたり点数 =
  所定点数（疾患×施設基準区分）
  ＋ 早期リハ加算（入院 かつ 入院日から14日以内 → 3日以内:60 / 4日目以降:25）
  ＋ 初期加算（発症等から14日以内 かつ 届出施設 → 45）
  ＋ 急性期リハ加算（入院 かつ 発症等から14日以内 かつ 対象患者 → 50）
  ＋ 休日リハ加算（土日祝 かつ 発症等から30日以内 → 25）
  － 離床なし減算（新設区分を適用）
※ 1日累計が上限単位（6 or 9、運動器Ⅰは6）を超えたら算定不可
```
